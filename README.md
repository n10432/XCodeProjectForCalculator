# XCodeProjectForCalculator
For code camp

[アプリ概要]

計算する際に暇つぶし用のミニゲームができるアプリ。

何かしらのボタンを押すと、画面上に表示されているモンスターを攻撃できる。

計算結果や押すボタンによってダメージや攻撃の種類が変わる。

モンスターを倒すと新しいモンスターが出現する。

[進捗]

・電卓部分の実装完了
